# Luna Market

This repository stores Luna Panel market data only.

It may contain:

- `registry.json`
- website navigation JSON files
- component, plugin, and theme manifests
- icon assets used by market entries

It must not contain Luna Panel application source code, secrets, databases, backups, or Docker build artifacts.

Registry URL:

```text
https://raw.githubusercontent.com/finallySmile/luna-market/main/registry.json
```
